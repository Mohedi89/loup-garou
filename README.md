# 🐺 الذياب والقرية — Loup-Garou

لعبة الذياب والقرية، تعمل كـ PWA — يمكن تثبيتها على الهاتف وتشغيلها بدون إنترنت.

## نشر على GitHub Pages

1. **ارفع الملفات** على GitHub في repository جديد
2. اذهب إلى **Settings → Pages**
3. اختر **Source: Deploy from a branch** ثم `main` / `root`
4. اضغط **Save** — بعد دقيقة التطبيق يكون متاح على:
   `https://[اسمك].github.io/[اسم-الrepository]/`
